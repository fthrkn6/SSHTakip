Mevcut Servis_Durumu.xlsx dosyalarındaki tüm verileri veritabanına aktarır. RCA analizi bu verilerden yapılır.

Bu işlem mevcut Excel dosyalarını değiştirmez, sadece DB'ye kopyalar. Notlardaki (Sistem/Alt Sistem/Açıklama) bilgileri de aktarılır.
Geçmiş Excel Yükle

Geçmiş dönemlere ait bir Excel dosyasını yükleyin. Veriler hem DB'ye hem Servis_Durumu.xlsx'e yazılır.

Excel Formatı:
  • Satır 1 (Header): Tarihler (YYYY-MM-DD)
  • Sütun A: Tramvay/Araç IDleri
  • Hücre: ✓ / ✗ / ⚠ veya metin (Servis / Servis Dışı / İşletme...)
  • Hücre Notu (opsiyonel): Sistem: X / Alt Sistem: Y / Açıklama: Z
Yükleme Formatı Örneği
Araç 2026-01-15 2026-01-14 2026-01-13
1531 ✓ ✓ ✗ (not)
1532 ⚠ (not) ✓ ✓
✓ Yeşil = Serviste  |  ✗ Kırmızı = Servis Dışı  |  ⚠ Turuncu = İşletme Kaynaklı Servis Dışı
Servis Dışı ve İşletme hücrelerine Excel notu ekleyerek Sistem/Alt Sistem/Açıklama bilgisi yazabilirsiniz.
